# Решение проблемы "Ошибка 500" при проверке ответов

## Проблема обнаружена

При проверке ответов возникает ошибка 500. Причина: **закончилась квота/средства на аккаунте OpenAI**.

## Решение

### Вариант 1: Пополнить баланс OpenAI (РЕКОМЕНДУЕТСЯ)

1. Перейдите на https://platform.openai.com/account/billing
2. Войдите в свой аккаунт
3. Пополните баланс
4. После пополнения система заработает автоматически

### Вариант 2: Создать новый API ключ

1. Перейдите на https://platform.openai.com/api-keys
2. Создайте новый API ключ
3. Обновите файл `.env`:
   ```
   OPENAI_API_KEY=ваш_новый_ключ_здесь
   ```
4. Перезапустите сервер

### Вариант 3: Использовать другой API (требует изменения кода)

Если хотите использовать другой AI API (например, Anthropic Claude, Google Gemini), потребуется изменить код в `main.py`.

## Проверка квоты

Запустите скрипт для проверки:
```bash
python test_openai_key.py
```

Если увидите ошибку "insufficient_quota" - нужно пополнить баланс.

## После решения проблемы

1. Убедитесь, что баланс пополнен
2. Перезапустите сервер: `python main.py`
3. Попробуйте проверить ответ снова
