СПОСОБ 1: Через файл run_server.bat (РЕКОМЕНДУЕТСЯ)
───────────────────────────────────────────────────────────────
1. Дважды кликните на файл: run_server.bat
2. Дождитесь сообщения "Запуск сервера на http://localhost:8000"
3. Откройте браузер и перейдите на: http://localhost:8000
4. НЕ ЗАКРЫВАЙТЕ окно с сервером!


СПОСОБ 2: Через командную строку
───────────────────────────────────────────────────────────────
1. Откройте командную строку (cmd) или PowerShell
2. Перейдите в папку проекта:
   cd C:\Users\Gazabowow\Desktop\clinical-ai-backend
3. Запустите сервер:
   python main.py
4. Дождитесь сообщения о запуске
5. Откройте браузер: http://localhost:8000


ПРОВЕРКА РАБОТЫ СЕРВЕРА
───────────────────────────────────────────────────────────────
Если видите ошибку "localhost отказано в подключении":
1. Убедитесь, что сервер запущен (окно с Python не закрыто)
2. Проверьте, что порт 8000 свободен
3. Попробуйте другой браузер
4. Проверьте файрвол/антивирус (может блокировать)


ОСТАНОВКА СЕРВЕРА
───────────────────────────────────────────────────────────────
Нажмите Ctrl+C в окне, где запущен сервер


ВОЗМОЖНЫЕ ПРОБЛЕМЫ
───────────────────────────────────────────────────────────────
1. "ModuleNotFoundError" - установите зависимости:
   pip install -r requirements.txt

2. "OPENAI_API_KEY not found" - проверьте файл .env

3. Порт занят - измените порт в main.py (строка 259):
   uvicorn.run(app, host="0.0.0.0", port=8001)
