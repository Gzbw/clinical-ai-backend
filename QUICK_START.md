# Быстрый старт

## Вариант 1: Автоматический запуск (Windows)

Просто дважды кликните на файл `START.bat` - система автоматически:
- Создаст .env файл если его нет
- Установит зависимости если нужно
- Запустит сервер

Затем откройте браузер: http://localhost:8000

## Вариант 2: Ручной запуск

### Шаг 1: Установите зависимости
```bash
pip install -r requirements.txt
```

### Шаг 2: Создайте .env файл (если его нет)
```bash
copy Open_ai.env .env
```

### Шаг 3: Запустите сервер
```bash
python main.py
```

### Шаг 4: Откройте браузер
Перейдите на: http://localhost:8000

## Как использовать

1. **Выберите задачу** из списка (1-21)
2. **Прочитайте условие задачи**
3. **Введите ваш ответ** в поле (включите все 3 задания)
4. **Нажмите "Проверить ответ"**
5. **Изучите результаты**:
   - Общая оценка
   - Проверка по 6 критериям
   - Что сделано правильно
   - Что нужно улучшить

## Остановка сервера

Нажмите `Ctrl+C` в окне терминала
