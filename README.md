# Система проверки клинических задач с использованием ИИ

Веб-приложение для проверки ответов студентов по клиническим задачам с использованием искусственного интеллекта.

## Возможности

- ✅ Загрузка и хранение множественных клинических задач
- ✅ Выбор задачи из списка
- ✅ Проверка ответов студентов по 6 критериям оценки
- ✅ Детальная обратная связь с указанием ошибок и пропущенных элементов
- ✅ Общая оценка и оценка по каждому критерию
- ✅ Поддержка ответов, написанных своими словами

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Создайте файл `.env` в корневой директории и добавьте ваш API ключ OpenAI:
```
OPENAI_API_KEY=your_api_key_here
```

3. Добавьте задачи в файл `data/tasks.json` (см. инструкцию в `data/FORMAT_INSTRUCTIONS.md`)

## Запуск

```bash
python main.py
```

Сервер запустится на `http://localhost:8000`

Откройте браузер и перейдите по адресу `http://localhost:8000`

## Использование

1. Выберите задачу из выпадающего списка
2. Изучите условие задачи
3. Введите ваш ответ в текстовое поле (ответ должен включать все три задания)
4. Нажмите кнопку "Проверить ответ"
5. Просмотрите детальную оценку по каждому из 6 критериев

## Критерии оценки

Система проверяет ответы по следующим критериям:

1. **Ведущие синдромы заболевания** - выявление и описание основных синдромов
2. **Эпидемиологические данные** - выделение релевантных эпидемиологических факторов
3. **Характерные объективные данные** - определение значимых объективных признаков
4. **Обоснование диагноза** - правильная формулировка и обоснование предварительного диагноза
5. **План диагностики** - назначение необходимых обследований
6. **План лечения** - составление комплексного плана лечения

## Формат данных

Задачи хранятся в файле `data/tasks.json` в формате JSON. Подробная инструкция по формату находится в файле `data/FORMAT_INSTRUCTIONS.md`

## Структура проекта

```
clinical-ai-backend/
├── data/
│   ├── tasks.json              # Файл с клиническими задачами
│   └── FORMAT_INSTRUCTIONS.md  # Инструкция по формату данных
├── static/
│   ├── index.html              # Главная страница
│   ├── script.js               # JavaScript для фронтенда
│   └── style.css               # Стили CSS
├── main.py                     # Backend (FastAPI)
├── requirements.txt            # Зависимости Python
└── README.md                   # Этот файл
```

## Технологии

- **Backend**: FastAPI, Python
- **Frontend**: HTML, CSS, JavaScript
- **AI**: OpenAI GPT-4 (можно использовать GPT-3.5-turbo)
- **Хранение данных**: JSON файлы

## Примечания

- Для работы требуется API ключ OpenAI
- Рекомендуется использовать GPT-4 для более точной оценки
- Все ответы проверяются с учетом возможности формулировки своими словами
- Система указывает конкретные ошибки и пропущенные элементы
