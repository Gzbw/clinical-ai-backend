# Инструкция по запуску системы

## Шаг 1: Установка зависимостей

Убедитесь, что все зависимости установлены:

```bash
pip install -r requirements.txt
```

## Шаг 2: Настройка API ключа OpenAI

Убедитесь, что файл `.env` существует и содержит ваш API ключ:

```
OPENAI_API_KEY=ваш_ключ_здесь
```

Если файла `.env` нет, скопируйте его из `Open_ai.env`:
- Windows: `copy Open_ai.env .env`
- Linux/Mac: `cp Open_ai.env .env`

## Шаг 3: Запуск сервера

Запустите сервер командой:

```bash
python main.py
```

Или с uvicorn напрямую:

```bash
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

После запуска вы увидите сообщение, что сервер работает на порту 8000.

## Шаг 4: Открытие в браузере

Откройте браузер и перейдите по адресу:

```
http://localhost:8000
```

## Использование системы

1. **Выберите задачу** из выпадающего списка вверху страницы
2. **Изучите условие задачи** - оно отобразится ниже
3. **Введите ваш ответ** в текстовое поле (должен включать все три задания)
4. **Нажмите "Проверить ответ"**
5. **Просмотрите результаты** - вы увидите:
   - Общую оценку (0-100 баллов)
   - Детальную проверку по каждому из 6 критериев
   - Что сделано правильно
   - Что нужно улучшить

## Структура проекта

- `main.py` - Backend сервер (FastAPI)
- `static/index.html` - Главная страница
- `static/script.js` - JavaScript для фронтенда
- `static/style.css` - Стили CSS
- `data/tasks.json` - Файл с клиническими задачами (21 задача)
- `.env` - Файл с API ключом OpenAI

## Устранение проблем

### Ошибка "ModuleNotFoundError"
Установите зависимости: `pip install -r requirements.txt`

### Ошибка "OPENAI_API_KEY not found"
Проверьте, что файл `.env` существует и содержит правильный ключ

### Сервер не запускается
Убедитесь, что порт 8000 свободен. Можно изменить порт в `main.py` в строке:
```python
uvicorn.run(app, host="0.0.0.0", port=8000)
```

### Задачи не отображаются
Проверьте, что файл `data/tasks.json` существует и содержит задачи
